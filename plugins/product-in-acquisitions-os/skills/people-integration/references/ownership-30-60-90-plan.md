# 30 / 60 / 90 Ownership Plan (Per Acquired Product Leader)

Treat the first 90 days as a product launch for the acquired team's belonging, ownership, and influence. Run this plan per person.

## Days 1 to 30: Connection and Context

Goal: Belonging, not assimilation.

| Outcome | Signal of completion | Owner | Status |
|---------|---------------------|-------|--------|
| Mentor relationship established | First three syncs held | Direct manager | |
| Decision map session held | Written summary sent and confirmed | Direct manager | |
| Shadow rotations scheduled | All four shadow sessions on calendar | Direct manager | |
| One meaningful contribution to a real decision | The contribution is named publicly: "We are doing X because [name] surfaced Y" | Integration Owner | |
| Combined product narrative reviewed | Acquired leader can repeat it without improvising | Direct manager | |

## Days 30 to 60: Early Ownership

Goal: Real scope, with enough support to succeed. This is not a test.

| Outcome | Signal of completion | Owner | Status |
|---------|---------------------|-------|--------|
| Named ownership of a meaningful scope | Written scope statement, agreed by both parties | Direct manager | |
| First measurable outcome delivered or progressed | Specific evidence | Acquired leader | |
| Visible influence on the merged roadmap | Named contribution recorded in the roadmap rationale | Integration Owner | |
| Cross-functional working relationships established | At least three named peer relationships outside their function | Acquired leader | |

## Days 60 to 90: Integration into Rhythm

Goal: Combined team is stronger for having them.

| Outcome | Signal of completion | Owner | Status |
|---------|---------------------|-------|--------|
| Participates in the planning cycle as a peer | Named contributions in the latest planning doc | Integration Owner | |
| Builds peer relationships that sustain past integration | At least one named peer they trust enough to disagree with | Acquired leader | |
| Holds a perspective that has changed an outcome | One named example | Integration Owner | |
| Day 90 reflection | Direct manager + Integration Owner + acquired leader hold a 30-minute reflection | All | |

## Day 90 question

> "Is the combined team stronger for having them?"

If the answer is not clearly yes, the plan has not worked yet, and the next 30 days must be spent making it work.
